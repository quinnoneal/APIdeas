APIdea Documentation

Currently the webservice consists of three functions: receiving data for a specific APIdea, all APIdeas, or a random APIdea, and a function to add your own APIdeas. An example of what the response body would look like can be seen below.

To receive the shown data, you have to make a HTTP GET call on the resource /apideas/{ID} or /apideas or /apideas/random of this website's IP. To add an APIdea, make a HTTP POST call on the resource /apideas with the data of an APIdea in JSON format in the request body. An ID is required.

GET /apideas/random
